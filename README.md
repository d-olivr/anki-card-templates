A collection of my custom Anki card templates. Each one lives in its own folder with its own `README` and previews. I made them for fun, mostly for myself. So they lean toward my own aesthetic, but feel free to tweak them to fit yours (..◜ᴗ◝..)

<br>

## 🟢 Templates

| Template | Description |
|---|---|
| 🍃 [Dark Leaf](./dark-leaf-card-template) | Simple front/back/e.g. layout with a collapsible example section, inline code, and code block theming. |
| 🍀 [Clover](./clover-card-template) | Front/back layout with a session topbar, decorated bubbles, hint-based example, MathJax/KaTeX support, and cloze styling. |

<br>

## 📗 Installation

Each template is installed independently, check the `README` inside each folder for the full steps.
